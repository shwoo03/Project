## 3. 깃허브 레포 업데이트 추적
- 특정 깃허브 레포에 tags를 추적하여 새로운 업데이트가 존재하면 디스코드 웹 훅으로 알려주는 기능 
<br>

- K-CTF 사이트에서 새로운 대회가 등록되면 디스코드 웹훅으로 알림을 보냅니다.
- 하루 1회 실행되며, 중복 알림을 방지합니다.
<br>
